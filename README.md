# 📘 Student Activity Tracker (Level-2 Hackathon)

## 📌 Project Overview
This project is an interactive **Student Activity Tracker** developed as part of a Front-End Development Hackathon (Level-2).

The application allows users to manage activities by marking them as completed and tracking progress in real time using JavaScript.

---

## 🎯 Objective
To implement front-end interactivity by enabling users to update activity status and view real-time progress without using any backend services.

---

## ⚙️ Features
- 📋 Display list of activities dynamically
- ✅ Mark activities as "Completed"
- 🔄 Real-time UI updates without page reload
- 📊 Live progress tracking (e.g., "4 out of 6 completed")
- 🎨 Clean UI with dark theme (black + blue accent)

---

## 🧠 Concepts Used
- JavaScript DOM Manipulation
- Event Handling
- State Management (Array-based)
- Dynamic Rendering

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
